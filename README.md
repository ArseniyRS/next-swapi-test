## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Formatting Eslint
```bash
npm run lint
# or
yarn lint
```

```bash
npm run stylelint # npm run stylelint:scss
# or
yarn stylelint # yarn stylelint:scss
```
